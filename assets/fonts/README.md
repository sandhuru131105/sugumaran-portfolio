# At Aero Font Integration

To display the custom **At Aero** font on this portfolio website, please place your licensed font files in this folder.

## File Names Expected:
- **Regular Weight (400):** `At-Aero-Regular.woff2` (or `At-Aero-Regular.woff`)
- **Bold Weight (700):** `At-Aero-Bold.woff2` (or `At-Aero-Bold.woff`)

## Where to get the font:
At Aero is a premium commercial font designed by Pedro Arilla. You can obtain a license and download the files from [Arillatype.Studio](https://arillatype.studio/fonts/at-aero).

*If you place files with different names, remember to update their paths in [style.css](file:///c:/Users/acer/Desktop/Projects/Sandhuru-portfolio/sugumaran-portfolio/assets/css/style.css).*
